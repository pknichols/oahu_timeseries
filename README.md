# Raw metabarcoding sequence data and metadata for O'ahu timeseries study
[![DOI](https://zenodo.org/badge/1054827145.svg)](https://doi.org/10.5281/zenodo.17099779)

Nichols PK & Marko PB (_in prep._). Spatial complexity, not temporal noise, structures eDNA community profiles on coral reefs.  

This repository contains raw environmental DNA (eDNA) metadata and metabarcoding sequence data. The dataset includes:

* FAIRe-compatible sample and PCR metadata (.xlsx)
* Illumina MiSeq raw sequence data (.fastq.gz)
